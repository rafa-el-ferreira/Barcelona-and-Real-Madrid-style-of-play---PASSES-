This project analyzes and compares the home playing styles of FC Barcelona and Real Madrid using detailed event‑level passing data. The goal was to understand how both teams build play, progress the ball, and maintain possession when playing at home.

The analysis includes:

Filtering and isolating all passes, accurate passes, forward passes, and accurate forward passes for each team

Handling dynamic team_id values across matches to correctly assign events

Computing key passing metrics such as total passes, forward passes, pass accuracy, and forward pass accuracy

Merging all metrics into a clean comparative DataFrame

Interpreting the results from a tactical perspective

Conclusion:  
Despite their traditionally contrasting playing philosophies — Barcelona’s structured positional play vs. Real Madrid’s more vertical and transition‑oriented approach — the passing metrics at home were surprisingly similar across all categories. Both teams showed comparable volume, accuracy, and forward progression patterns, suggesting that home context may reduce stylistic differences and lead to more balanced passing profiles.

This project provides a reproducible framework for team‑level passing analysis and highlights how data can challenge or confirm tactical assumptions.
